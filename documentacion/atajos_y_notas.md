Atajos de teclado en IntelliJ IDEA:

Ctrl + Shift + P: Abre la paleta de comandos, donde puedes buscar y ejecutar comandos rápidamente.
Ctrl + s: Guarda el archivo actual.
psvm + Tab: Genera el método main en Java.
Alt + ctrl+ s: Abre la configuración de IntelliJ IDEA, donde puedes personalizar tus preferencias y ajustes.

Si vas a Edit -> Comment with Line Comment, el atajo es Ctrl + / para comentar o descomentar una línea de código.
Si vas a Edit -> Comment with Block Comment, el atajo es Ctrl + Shift + / para comentar o descomentar un bloque de código.
